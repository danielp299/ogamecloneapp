# Large Shield Dome (LSD)

A massive planetary shield.

## Technical Details
- **Base Cost:** Metal: 50,000, Crystal: 50,000
- **Base Duration:** 20m
- **Requirements:** Shipyard (Level 6), Shielding Technology (Level 6)

## Stats
- **Structure:** 100,000
- **Shield:** 10,000
- **Attack:** 0

## Functionality
Provides an extremely strong shield for the planet. Only one can be built per planet.
