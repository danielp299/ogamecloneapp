# Small Shield Dome (SSD)

Protects the planet with a shield.

## Technical Details
- **Base Cost:** Metal: 10,000, Crystal: 10,000
- **Base Duration:** 5m
- **Requirements:** Shipyard (Level 1), Shielding Technology (Level 2)

## Stats
- **Structure:** 20,000
- **Shield:** 2,000
- **Attack:** 0

## Functionality
Provides a strong shield for the planet. Only one can be built per planet.
