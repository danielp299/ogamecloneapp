# Módulo: Factory (Fábrica/Astilleros)

## Descripción General
El módulo Factory, también conocido como Shipyard, es donde se construyen naves y defensas. Acelera producciones con niveles altos, y es crucial para flotas y defensas. Incluye fábricas de robots y nanitas para optimización.

## Características Principales
- **Edificios Relacionados**:
  - **Robotics Factory**: Reduce tiempos de construcción de edificios (no defensas/naves).
  - **Shipyard**: Construye naves y defensas; reduce tiempos de ambas.
  - **Nanite Factory**: Halva tiempos de construcción para todo (edificios, naves, defensas) por nivel.
- **Producción**: Cola secuencial; no cancelable. Tiempos: Tiempo = (Metal + Cristal) / (2500 * (1 + Shipyard) * 2^Nivel Nanite * Velocidad Universo).
- **Requisitos**: Robotics 2 para Shipyard; Robotics 10 y Computer 10 para Nanite.

## Mecánicas de Juego
- **Eficiencia**: Nanites son más potentes que Robotics para altos niveles.
- **Bloqueo**: No se pueden construir naves/defensas mientras se mejora Nanite.
- **Costos**: Duplican por nivel; altos para Nanite.

## Estrategias Recomendadas
- Construye Robotics primero para acelerar.
- Prioriza Shipyard para flotas/defensas.
- Invierte en Nanite tarde: Caro, pero ahorra tiempo a largo plazo.
- Balancea: No descuides otros edificios por fábricas.

Este módulo impulsa la producción militar y económica.
