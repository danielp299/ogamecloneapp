# Módulo: Constellation (Constelación/Galaxia)

## Descripción General
El módulo Constellation, o Galaxy View, muestra el mapa galáctico, permitiendo explorar sistemas solares, espiar planetas y planificar ataques/colonizaciones. Es esencial para inteligencia y navegación.

## Características Principales
- **Vista por Sistema**: Muestra 15 slots por sistema: planetas, lunas, debris.
- **Información Visible**: Propietario, estado (activo/inactivo), alianzas (si se espía).
- **Acciones**: Enviar flotas, espiar, phalanx (desde lunas).
- **Coordenadas**: Galaxia:Sistema:Posición (ej. 1:123:4).
- **Distancias**: 20,000 unidades entre galaxias; universos circulares conectan extremos.

## Mecánicas de Juego
- **Espionaje**: Revela más info con niveles altos de Espionage Technology.
- **Colonización**: Busca slots vacíos (Outer Space).
- **Debris Fields**: Visibles; recolectables.
- **Herramientas Externas**: Mapas comunitarios para vistas globales.

## Estrategias Recomendadas
- Explora temprano: Encuentra inactivos para farming.
- Marca objetivos: Usa bookmarks para accesos rápidos.
- Evita detección: Espía antes de atacar.
- Navega eficientemente: Planifica rutas para minimizar tiempos.

Este módulo es clave para estrategia espacial y scouting.
